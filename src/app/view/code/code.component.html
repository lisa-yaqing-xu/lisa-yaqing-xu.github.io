Animated Poseur

A frame animation-focused multi-layer drawing application that initially started as my senior project. Is currently built on the AngularJS framework with JavaEE backend using Google App Engine hosting and database, but I have plans to port it to Angular2+ with Spring Boot in the near future for more up-to-date technology and better maintainability before I implement more features.

Chan Convex Hull Algorithm Implementation

An implementation of Timothy Chan's convex Hull algorithm, coded and visualized in JavaScript.

Piggy Bank Polymer Components

This is an example of a resource management tool we began to build. It leveraged an existing time tracking solution that we implemented previously to pull in the hours recorded by each resource. It was our intent to perform some calculations with the data collected by PiggyBank and the other platform to enable financial reports to be created. The reports would have included organizational performance against quarterly and annual budget targets, as well as resource utilization metrics.

PiggyBank is built using the Google Polymer framework. The charts were built using D3.js.
